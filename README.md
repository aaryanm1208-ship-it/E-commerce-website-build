# Amazon Lite - Full Stack E-commerce Website

A **full-stack Amazon-like e-commerce platform** built with **React, Node.js, Express, and MongoDB**.  
Supports user authentication, product management, cart, orders, and admin panel.

## Features

- User Authentication (Register/Login)
- Product Listing & Details
- Cart Management (Add, Remove, Update)
- Order Placement & History
- Admin Panel (Manage Products & Orders)
- Search, Filter & Sort Products
- Simple, Clean UI

## Tech Stack

- Frontend: React.js, CSS / Tailwind
- Backend: Node.js, Express.js
- Database: MongoDB
- Auth: JWT + bcrypt
- Tools: VS Code, Postman
